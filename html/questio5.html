Text Formatting Tags in HTML

Bold Text: This is bold text using the <b> tag.

Strong Text: This is important text using the <strong> tag.

Italic Text: This is italicized text using the <i> tag.

Emphasized Text: This is emphasized text using the <em> tag.

Short Quotation: This is a quoted text using the <q> tag.

Blockquote Example:
“The only limit to our realization of tomorrow is our doubts of today.”

Deleted Text: This is deleted text using the <del> tag.

Inserted Text: This is inserted text using the <ins> tag.

Fixed-width text using <tt> (now deprecated in HTML5).

            This is preformatted text.
            It preserves spacing and line breaks.

Code Example:

<html> <body> <h1>Hello, World!</h1> </body> </html>

HTML Entities: © ® < > & € £ ¥

Chemical Formula Example: H2O (Water)

Mathematical Power Example: X2 + Y2 = Z2
